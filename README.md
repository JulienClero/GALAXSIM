# GALAXSIM
My attempt to make a N-Body simulation running in Python with cuda.

## Controls
*GALAXSIMr* uses a minimalistic controls.
| Key/Input | Action |
| :---: | :---: |
| LMB | Place a Star |
| RMB | Places a Galaxy |
| LMB or RMB + drag | Gives bodies initial velocities |
| ESC | Close GALAXYM|
